# GTM:OS Quick Start

5-block quick onboarding. Get running in minutes. Remaining fields use defaults from defaults.md.

## Block 1 — Offer
"What do you sell? One sentence."
Example: "AI-powered sales automation for B2B SaaS companies"

Save to: `workspace.config.md` and `ICP.md`

## Block 2 — Target
"Who's your ideal customer? Give me a title and company type."
Example: "VP Sales at B2B SaaS companies, 50-500 employees"

Save to: `ICP.md` and `PERSONA.md`

## Block 3 — Pain
"What's the #1 problem you solve?"
Example: "SDRs spend 4 hours/day on manual prospecting instead of selling"

Save to: `ICP.md` (pain section)

## Block 4 — Angle
"Why you, why now? One sentence."
Example: "We're the only tool that combines intent data with AI personalization"

Save to: `ICP.md` (angle section)

## Block 5 — Voice
"How should I sound? Pick one: formal, casual, provocative, consultative."
Default: consultative

Save to: `TOV.md`

---

## After quick start:

1. Pre-fill all remaining workspace files with defaults from defaults.md:
   - TOOLS.md, INFRASTRUCTURE.md, PIPELINE.md, BOOKING.md
   - COMPETITORS.md, SUPPRESSION.md, MULTICHANNEL.md
   - LEARNINGS.md, ROADMAP.md, COSTS.md

2. Display: "Quick start complete. Here's what I set up:"
   - Show populated files with key values
   - Show files using defaults (can be refined later)

3. Suggest next step based on role:
   - SDR: `/gtm:new-campaign`
   - GTM Engineer: `/gtm:infra`
   - Head of Sales: `/gtm:dashboard`
   - Founder: `/gtm:research`
   - Agency: `/gtm:onboard {next-client}`
