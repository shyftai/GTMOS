# Command: Write Outbound Sequence

## Trigger
"Write a [N]-touch sequence" or "Draft the outbound sequence for [campaign]"

## Steps
1. Load ICP.md, PERSONA.md, and TOV.md from active workspace; load BRIEFING.md from active campaign
2. Load campaign.config.md for channel and touch count
3. Check context/research/signal-angles.md for relevant angles to incorporate
4. Draft each touch in order:
   - Touch 1: cold open, hook-led, single CTA
   - Touch 2: light nudge, different angle
   - Touch 3: value add or social proof
   - Touch 4 (if applicable): low-pressure close
5. Apply TOV.md channel-specific rules (email vs LinkedIn)
6. After drafting: run validate-copy automatically
7. Present with inline flags for anything borderline
8. Save approved version to copy/approved/
