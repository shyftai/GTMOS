## What makes these work

| Principle | How it shows up |
|-----------|----------------|
| Peer voice | Reads like a colleague, not a marketer |
| Observation-led | Every touch 1 opens with something about THEM |
| Ruthless brevity | Touch 1: 52-68 words. Follow-ups: 22-44 words |
| Interest-based CTA | "Worth a look?" / "Curious?" / "Make sense to chat?" |
| Angle rotation | Every touch uses a different angle |
| Subject lines | 2-4 words, lowercase, no punctuation |
| No spam words | Zero "excited", "thrilled", "game-changing" |
| Specificity | Concrete numbers, named companies, real signals |
