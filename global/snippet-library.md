# Snippet Library

Reusable copy fragments across workspaces. These grow over time as campaigns run and winning copy is identified. Tag each snippet with the context where it works best.

---

## Follow-up angles

Use these to rotate angles across touches — never repeat the same pitch.

| Angle | Snippet | Best for touch |
|-------|---------|---------------|
| Different pain | Separate from the [topic] note — curious how {{company}} handles [different pain]. | Touch 2 |
| Social proof | [Customer] had the same [challenge] — ended up [outcome]. Thought it might resonate. | Touch 2-3 |
| Industry insight | Seeing a pattern across [industry] — [insight]. {{company}} might be running into this too. | Touch 2-3 |
| Value add | Found this [benchmark/report/insight] — thought it might be useful regardless. | Touch 3 |
| Breakup | Figured this isn't a priority right now — no worries. If it comes up later, happy to help. | Touch 4 |
| Timing check | Revisiting this from [X weeks] ago — has anything changed on the [pain] front? | Re-engagement |

## Objection responses

| Objection | Response snippet | Notes |
|-----------|-----------------|-------|
| "We already use [competitor]" | Totally fair — most of the teams we work with were on [competitor] before. The gap they kept running into was [specific gap]. Worth comparing? | Acknowledge, don't attack |
| "Not a priority right now" | Makes sense. When it does come up, happy to help. Mind if I check back in [timeframe]? | Leave door open, set follow-up |
| "Send me more info" | Absolutely — what specifically would be most useful? [Feature area], [use case], or a quick case study? | Qualify the ask, don't dump a brochure |
| "Too expensive" | Understood. Most teams offset the cost by [specific ROI mechanism]. Would it help to see the math on that? | Reframe to value, not price |
| "We're locked into a contract" | Got it — when does that come up for renewal? Happy to circle back then with a comparison. | Time the follow-up |

## How to use

- When writing a sequence, scan this library for relevant snippets as starting points
- Adapt to the workspace's TOV.md — these are templates, not copy-paste
- After a campaign, add winning copy to this library with performance notes
- Tag snippets by channel and best use case so they're easy to find
- Remove snippets that consistently underperform
